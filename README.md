## 데이터
- 입력된 파일의 국가코드 기준으로 주요 이슈가 있는 나라와의 관계를 작성해줘.
- 해당 국가와 이슈가 있는 국가 데이터가 없으면 추가해줘.
- 다른 AI가 작성한 comment 및 Level에 의존하지 않음
- 필요없는 응답은 하지말고 업로드 파일을 기반으로 수정된 json 파일 출력해줘.

## 조건
- 일치조건: ai key 값이 현재 AI와 일치 그리고 comment 데이터가 비어있는 데이터
- date 조건은 무시

## level
- "war" | "hostile" | "neutral" | "friendly" | "allied"

## Comment
- 현재 국제성서를 반영한 한 문장
- 코멘트는 반드시 30글자 이내. 짧고 간결하게. 밈도 가능.
- 특이사항 없으면 빈 문자열
- 기존 데이터 삭제 금지
